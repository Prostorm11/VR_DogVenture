VR Word Puzzle Game

A Virtual Reality (VR) word puzzle game developed in Unity Hub, where players solve increasingly difficult word challenges while interacting with an emotionally reactive dog companion.

Project Overview

The VR Word Puzzle Game is an immersive educational puzzle experience built for VR headsets. Players create meaningful words from a given base word to score points and advance through levels.
